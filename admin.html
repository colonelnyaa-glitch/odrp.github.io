編集方法

右側のプレビュー内で、変更したい文字をクリックして直接入力します。

  1. 文字をクリックして編集
  2. リンク・画像は要素をクリック
  3. 「更新HTMLを書き出す」
  4. GitHubのindex.htmlと交換

選択中の要素

リンクまたは画像をクリックしてください。

保存・書き出し

注意

GitHub Pagesは静的サイトのため、この管理画面からGitHubへ直接保存はできません。書き出したindex.htmlをGitHubへアップロードしてください。
